# IR-Thesis
Regressions, data cleaning for international relations senior thesis

The main regressions used in this project can be found in Master_File.do. 

Labels for my data are created in the labels.do files, actual datasets used in the labelling are referenced in my paper. 

Replication_Ideal_Votes.do is a replication of techniques used by Erik Voeten to generate ideal point estimates of UN votes. 

Other files are manipulations used to clean and sort my data.
